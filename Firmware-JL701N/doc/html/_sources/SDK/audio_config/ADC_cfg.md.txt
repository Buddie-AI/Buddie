# ADC配置

## 音频配置
- **省电容MIC**：省电容mic使能
  - 注意：不用的时候建议关闭，可以节省优化代码体积
- **MICLDO电压**：设置MICLDO电压档位
  - 注意：为保证该电源的稳定性，IOVDD需要比MICLDO高200~300mV
- **性能模式**：设置ADC的性能模式
  - 低功耗模式
  - 高性能模式

  
## 节点配置
### MIC使能配置
- **MIC使能**
  - 设置MIC使能通道(跟据不同芯片可配置的通道数量也会有所不同)
### MICx 配置
- **MIC 模式**
  - 设置MIC的输入方式，需要跟据实际外部硬件电路进行选择
- **增益**
  - 设置MIC的输入信号增益。通过模拟增益调整MIC的输入信号大小
- **前级增益**
  - 设置MIC的输入信号前级增益
- **输入端口**
  - 设置MIC的输入端口。选择该路MIC从哪个IO输入信号
- **供电端口**
  - 设置MIC的供电方式。常见的供电方式分为MIC LDO、MIC BIAS、其他(IO或外部供电)
- **MIC BIAS电阻档位**
  - 设置MIC BIAS的电阻档位。当供电端口选择MIC BIAS供电时可通过配置该电阻档位调整MIC口的电压
- **IO供电选择**
  - 设置外部供电的IO。当供电方式选择IO供电时可通过该配置选择供电的IO口
- **DCC使能**
  - 设置是否打开DCC去直流的功能
- **DCC 截止频率**
  - 设置DCC滤除直流的截止频率
