# 编解码配置

## 解码格式
### 通用解码格式
|序号|格式|备注|
|---|---|---|
|1|WAV||
|2|MP3||
|3|WMA||
|4|FLAC||
|5|APE|仅支持Fast/Normal压缩等级|
|6|AAC||
|7|SBC||

### 自定义解码格式
自定义解码格式，通常是为了实现特定的功能需求，同时又兼顾资源效率而定制开发的编解码算法，比如提示音，常用的自定义格式汇总如下
|序号|格式|备注|
|---|---|---|
|1|F2A||
|2|WTG||
|3|MTY||
|4|WTS||
