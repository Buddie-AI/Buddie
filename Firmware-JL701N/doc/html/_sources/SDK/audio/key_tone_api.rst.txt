按键音
=============

按键音播放API是系统提供的用于播放按键提示音文件的接口,
每调用一次播放接口内部播放计数值加1, 播放结束后减1, 减到0停止播放。
按键音可以和包括提示音在内的其它音频叠加播放。

IO或AD类型的按键, 一般在按键按下的回调中发送播放按键音的消息，
触摸类型的按键，一般是在触摸驱动的按键音回调函数中发送播放按键音的消息。


play_key_tone_file
----------------------------------------
播放按键音文件, 参数为按键音文件的相对路径。可用".*"代替文件后缀名。

原型: 

.. code-block:: c

    int play_key_tone_file(const char *file_name)
    
示例:

.. code-block:: c

    play_key_tone_file("tone_en/key_tone.*");
