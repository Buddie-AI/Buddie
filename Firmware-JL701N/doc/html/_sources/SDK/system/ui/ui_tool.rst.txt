控件列表
~~~~~~~~~~~~~~~

ui 工具新建控件时采用拖动的形式，单击选中控件后拖动到 ui 设计窗口，控件灰色表示不可用，新建页面后必须先创建一个图层，图层上可以创建布局，布局上可以创建各类控件；目前的芯片只支持单图层，所以每个页面只能建立一个图层

控件属性
~~~~~~~~~~~~~~~

控件 ID 号：每个控件都有唯一的 ID 号，给控件的 api 接口使用；控件创建时会默认分配一个ID 号，可以根据需要修改

对齐方式：有三种对齐方式，分别为左对齐/居中对齐/右对齐，默认为居中对齐

位置坐标：在ui设计窗口拖动控件时，该控件的坐标信息会在属性栏实时更新，需要精确控制的场合，也可以手动输入；在选中控件的前提下也可以通过键盘的方向键进行微调

背景颜色：创建控件时会默认指定背景颜色，用不同的颜色加以区分不同的控件类型，若不需要背景颜色可点击右边的箭头去掉。

背景图片：可以根据需求指定控件的背景图片
