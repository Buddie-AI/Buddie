gpio配置
#######################

不漏电原则
-----------------
不漏电原则：数字输入 die/dieh 配置状态下，io 为输入状态（cpu/外设/crossbar），外部没有确
定电平有浮空漏电的危险，漏电电流不确定，请确保初始化外设/io 的状态，运行状态和低功耗模式
通用，一般将不使用的io配置为高阻状态。


配置
-----------------
sdk上电初始化：
1、在port_init处理io状态
2、将不使用的io配置为高阻

sleep模式：
1、在回调函数处理
2、确保初始化流程无问题，即不使用的io配置为高阻。

soff模式：
1、在回调函数处理
2、所有不使用的io默认配置为高阻，外设由库的程序进行释放
