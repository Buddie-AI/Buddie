soundbar 说明
###############

概述
-------
    SDK工程满足 2.0/2.1 声道soundbar应用开发，提供丰富的音源输入接口和灵活的音效调试接口，具体功能如下：

-   支持多种功能模式

    + SPDIF模式
    + BT模式
    + aux模式
    + 设备解码（USB/SD）模式
    + FM模式
    + 待机模式

-   支持多种音源输入

    + HDMI ARC
    + 光纤、同轴
    + AUX IN 
    + 蓝牙
    + USB、SD卡
    + FM

-   提供灵活音效调试接口和算法
-   支持多种解码格式：MP2/MP3/WAV/WMA/APE/FLAC等
-   支持NFC 从机应用，支持IOS/IEC-14443 Type-A第2,3部分协议，辅助蓝牙连接
-   可自由接入APP控制
