# 位宽配置说明
* 16bit/24bit位宽：音频采样样的位数，位数越多，精度越高
* 本SDK支持16bit、24bit位宽的选配

## 位宽配置介绍

* 音频流程图，可独立配置目标位宽

## 位宽配置注意事项

* 位宽选配不支持在线更新，选配后需重新编译下载
* 选16bit时，流程图内的节点可按照需求调整输入输出端点的位宽
* 选24bit时，流程图内的节点输出位宽无需变动，SDK运行时会自动把所有节点协商24bit位宽
* DRC节点在16bit与24bit位宽下，输入输出曲线的表达范围有变化，二者调试效果不能简单的平移使用。16bit满幅度是90.3dB，24bit的满幅度是138.47dB，如果想将16bit环境调好的DRC参数配置平移到24bit环境，可将16bit环境下的输入输出幅度分别加上16bit与24bit满幅度的差值48.17（138.47-90.3）
