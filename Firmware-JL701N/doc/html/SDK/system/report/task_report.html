17.1. 任务报告

任务在运行过程中会统计以下信息：

  • 1.处理器运行时长cpu run(unit:usec)

  • 2.当前压栈值StaskPush(unit:word)

  • 3.最深压栈值StackMax(unit:word)

  • 4.任务最新开始运行时刻SwInTime(unit:usec)

  • 5.任务运行时长RunTime(unit:usec)

  • 6.任务名Name

17.1.1. os_system_info_output

功能:

输出任务状态。

void os_system_info_output(void);

参数:

示例:

os_system_info_output();

17.1.2. os_system_info_reset

功能:

重置任务运行时长。

Warning

频率统计模块会定时重置，具体查阅clock_manager.c

void os_system_info_reset(void);

参数:

示例:

os_system_info_reset();
