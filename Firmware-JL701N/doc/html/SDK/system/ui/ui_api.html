显示数字

函数原型

int ui_number_update_by_id(int id, struct unumber *n);

功能描述

更新数字显示

参数说明

id：数字控件 ID
enum {
TYPE_NUM, //数字
TYPE_STRING, //数字字符串
};
struct unumber {
u8 numbs; //数字个数
u8 type; //类型
u32 number[2]; //数值，当 TYPE_NUM 时有效
u8 *num_str; //字符串,当 TYPE_STRING 时有效
};
n:需要显示的数字参数

输出

0：成功 其他：失败

例子

struct unumber n;
n.type = TYPE_NUM;
n.numbs = 1;
n.number[0] = 10;
ui_number_update_by_id(NUMBER, &n);
